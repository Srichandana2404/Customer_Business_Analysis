# Data Analytics Project

## Overview

This project demonstrates an end-to-end **data analytics workflow**, starting from data loading and cleaning to building an interactive dashboard and presenting insights. The goal of the project is to transform raw data into meaningful insights using Python, SQL, and Power BI.

The project includes:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Running SQL queries on a relational database
* Building an interactive Power BI dashboard

This project highlights practical data analytics skills that are commonly used in real-world business environments.

---

## Dataset

The dataset used in this project contains structured data suitable for analysis. It was initially processed in Python to clean and prepare it for further analysis.

Key steps performed on the dataset:

* Handling missing values
* Removing duplicates
* Formatting columns
* Preparing the dataset for SQL queries and visualization

After cleaning, the dataset was imported into a SQL database for further querying and analysis.

## Project Workflow / Steps

### 1. Data Loading

The dataset was loaded into Python using libraries such as:

* pandas
* numpy

### 2. Data Cleaning

The dataset was cleaned to ensure accuracy and usability. Cleaning steps included:

* Removing null values
* Fixing incorrect data types
* Eliminating duplicate records
* Preparing columns for analysis

### 3. Exploratory Data Analysis (EDA)

Basic exploratory analysis was performed to understand patterns and trends in the dataset. This included:

* Summary statistics
* Data visualization
* Identifying key insights

### 4. Database Integration

The cleaned dataset was imported into a SQL database. SQL queries were written to:

* Filter data
* Aggregate results
* Extract meaningful insights

### 5. Power BI Dashboard

Power BI was used to build an interactive dashboard to visualize the insights from the dataset.

Dashboard features include:

* Key metrics and KPIs
* Interactive charts and graphs
* Filters and slicers for dynamic analysis
